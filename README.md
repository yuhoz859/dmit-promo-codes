## DMIT 的三条退款和 IP 政策

- **3 天退款保障**：购买后 3 天内使用不超过 30GB 流量，可以全额退款。买之前不确定，先用 3 天测性能，没问题再放心续费

- **IP 被墙替换政策**：IP 被 GFW 封锁后，15 天内免费换一次。超过后每次收 $5。DMIT 官网有明确说明，不是"联系客服再说"的模糊政策

## 常见问题

**Q：优惠码每次续费都能享受折扣吗？**

能。DMIT 的循环折扣码（Recurring）就是字面意思，每次账单都按折扣价出，不是"首单优惠"陷阱。续费价格和买入时一样。

**Q：月付能用折扣码吗？**

大部分主力折扣码要求季付或年付才能生效，月付基本失效。如果只是短期测试，月付按原价买，测完觉得好再换年付+码续费。

**Q：LAX.Pro 和 LAX.EB 怎么选？**

电信用户选 Pro，CN2 GIA 三网回程优势对电信最明显。联通和移动用户用 EB 性价比更好——价格相近但流量更大，加上 8 折码后比 Pro 便宜不少。

**Q：香港 Pro 经常没货怎么办？**

HKG.Pro 是 DMIT 最热门的系列，补货后几小时内常卖完。一般在节假日前后（黑五、双十一、圣诞）放货概率较高。T1 系列库存相对宽松，可以先入手体验 DMIT 品质，再等 Pro 补货。

**Q：流量用完后会断线吗？**

不停机，直接降速。T1 系列超量后限速到 50Mbps 无限流量；Pro 和 EB 系列的超量降速幅度根据套餐不同，一般在 100Mbps 至 1Gbps 之间。不会突然断，只是速度变慢。
